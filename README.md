# Moikkabot 🎮

Desktop-sovellus Twitch-streamaajille:
- !jonoon komento
- OBS overlay
- Admin-paneeli
- Statsit (!stats)
- Desktop-app (Electron)

## 🔽 Lataus

👉 Mene **Releases**-välilehdelle ja lataa uusin `.exe`

## ⚙️ Käyttöönotto

1. Asenna sovellus
2. Lisää botille mod-oikeudet Twitchissä
3. Avaa sovellus
4. Syötä kanavan nimi Asetukset-välilehdellä
5. Valmis 🎉

## 💬 Komennot

- `!jonoon` – liity jonoon
- `!seuraava` – mod: seuraava pelaaja
- `!jono` – näytä jono
- `!stats` – omat tilastot
